# Simple Power BI Cookeys Dashboard

This repository contains a simple sales dashboard for a fictional cookie company. The project uses Microsoft Power BI to create visualizations based on data from an Excel spreadsheet.

## Project Files

*   `My_Cookeys_Company.pbix`: The Power BI report file. This file includes the data model, DAX measures, and all dashboard visuals.
*   `My_Cookie_Company_Data.xlsx`: The Excel workbook that serves as the data source for the Power BI report. It contains the raw sales and product data.

## How to Use

To explore the dashboard and its underlying data, follow these steps:

1.  **Prerequisites**: You must have [Microsoft Power BI Desktop](https://powerbi.microsoft.com/en-us/desktop/) installed on your machine.

2.  **Download**: Clone or download this repository to your local machine.

3.  **File Placement**: For the data source connection to work automatically, ensure that both `My_Cookeys_Company.pbix` and `My_Cookie_Company_Data.xlsx` are saved in the same folder.

4.  **View the Dashboard**: Open the `My_Cookeys_Company.pbix` file with Power BI Desktop. You can now interact with the dashboard, explore the different report pages, and examine the data model and DAX formulas.
